# ITI Projects

- src: contains all files for both projects
- src/lzw_coder_decoder: Project 1
  - Implements a coder and decoder using the  LZW algorithm.
  - Run: python main.py
- src/knn_classifier: Project 2
  - Implements a KNN classifier using the LZW coder as a distance metric for classification.
  - Run: python knn.py

#### Note: Navigate to the desired project before executing each script, the path for the files within the script may only resolve correctly this way.
